# 🚀 DecodeLabs — Frontend Development Project 1
### Static Webpage Design | Batch 2026

---

## 📌 Project Overview

This is **Project 1** of the DecodeLabs Industrial Training Program (Batch 2026).  
The goal is to build a clean, semantic, and accessible **static webpage** using only **HTML5 and CSS3** — no JavaScript, no frameworks.

This project proves the ability to:
- Structure content using semantic HTML
- Style a webpage using an external CSS file
- Apply CSS Grid and Flexbox layout systems
- Follow accessibility and W3C standards

---

## 🗂️ Project Structure

```
project-1/
│
├── index.html        ← Main HTML file (all page structure)
├── style.css         ← External stylesheet (all visual styling)
└── README.md         ← This file
```

> ⚠️ Both files must be in the **same folder** for the CSS to load correctly.

---

## 🌐 How to Run

1. Download both `index.html` and `style.css`
2. Place them in the **same folder** on your computer
3. Double-click `index.html` — it will open in your browser
4. No server, no installation, no setup needed!
